# Rongali Chaitanya Portfolio

Personal portfolio website for `rongalichaitanya.com`.

## Stack

- Next.js
- React
- TypeScript
- CSS
- Vercel-ready deployment

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Deployment

Push this repository to GitHub, then import it in Vercel. Add
`rongalichaitanya.com` in Vercel project settings and copy the DNS records that
Vercel gives you into Hostinger.
